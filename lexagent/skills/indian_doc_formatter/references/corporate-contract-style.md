# Corporate Contract Style — Indian Standard

## Page Setup
- Size: A4 (210mm × 297mm)
- Margins: Top 25mm, Bottom 25mm, Left 31.75mm, Right 25.4mm (court margin left)
- Orientation: Portrait

## Typography
- Body text: Times New Roman 12pt
- Heading 1 (Part/Chapter): Times New Roman 14pt Bold, ALL CAPS, centred
- Heading 2 (Clause group): Times New Roman 12pt Bold, Title Case
- Heading 3 (Sub-clause): Times New Roman 12pt Bold Italic
- Defined terms: Bold on first use, Title Case thereafter
- Recitals (WHEREAS): Times New Roman 12pt Italic, justified

## Spacing
- Body paragraphs: 1.15 line spacing, 6pt space-after
- Before Heading 1: 18pt; after: 12pt
- Before Heading 2: 12pt; after: 6pt
- Recitals: 1.0 line spacing, 6pt space-after

## Clause Numbering Scheme
```
1.        (Heading 2 — Clause Group)
1.1       (Body clause)
1.1.1     (Sub-clause)
(a)       (Paragraph)
(i)       (Sub-paragraph)
```
Use a single Word multi-level list so renumbering is automatic.

## Defined Terms
- Collected in a Definitions clause (typically Clause 1)
- Format: **"Term"** means [definition]. (bold quoted term, period at end)
- All subsequent uses: Title Case, no bold, no quotes

## Recitals
- Label: WHEREAS / AND WHEREAS
- Lettered: (A), (B), (C) if multiple
- Close with: NOW, THEREFORE, in consideration of the mutual covenants…

## Schedules / Annexures
- Page break before each Schedule
- Title style: Heading 1, "SCHEDULE [N] — [TITLE]"
- Body continues in same font/size as main document
- Schedule numbering restarts at 1 within each Schedule

## Signature Block
Two-column table (no visible borders), one column per party:
```
For and on behalf of              For and on behalf of
[PARTY A NAME]                    [PARTY B NAME]

________________________          ________________________
Name:                             Name:
Designation:                      Designation:
Date:                             Date:
Place:                            Place:
```

## Page Numbers
Footer, centred: "Page [X] of [Y]"
Font: Times New Roman 10pt

## Cover Page (if required)
```
[AGREEMENT TITLE]
Between
[PARTY A FULL NAME]
and
[PARTY B FULL NAME]

Dated: [DATE]
```
All centred, Heading 1 style for title, Body for parties.
