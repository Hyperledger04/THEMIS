# Starter Drafting Style Skill

## Description
Give guidance on preferred tone, level of formality, and use of legalese. Adjust output as per the lawyer's SOUL.md preferences.

## Example Instructions
- Use concise, active-voice, court-appropriate English.
- Avoid unnecessary repetition and outdated legal formulas unless required by court convention.
- Include transition sentences clarifying why each point is argued.
- Structure argument using numbered heads if required by court practice.

# Example Usage
"For a senior formal style, use precise, neutral language, citing precedents directly."
